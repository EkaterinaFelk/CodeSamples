# Snake Game

![App demo](<Screen Recording 2024-01-20 at 19.17.58.gif>)

Game on vanillaJS using es5 just for fun

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:8080`.

### Production build

```bash
npm run build
```

## Technology stack

-  JavaScript (es5)
-  scss
