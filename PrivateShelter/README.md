# Private Shelter

![shelter screenshot](public/screenshot.png?raw=true "Title")

Website for private shelter, where you can choose pet

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm run serve:api
```
```bash
npm start
```

You can view the development server at `localhost:3080`.

### Production build

```bash
npm run build
```

## Technology stack

-  JavaScript
-  Typescript
-  React
-  Redux
-  scss
